## 📋 Sobre o Projeto

O **TDB Responde** nasceu da necessidade de otimizar o atendimento da ONG Turma do Bem, que recebia comunicações de diversos canais (WhatsApp, e-mail, redes sociais) de forma desorganizada.

Este projeto é um **MVP (Produto Mínimo Viável)** desenvolvido como Challenge acadêmico, com o objetivo de centralizar e humanizar o atendimento aos assistidos.

### ✨ Funcionalidades

- 📱 **Site institucional** com informações sobre o projeto
- 👥 **Página de equipe** com integrantes e contatos
- ❓ **FAQ interativo** com perguntas frequentes
- 📬 **Formulário de contato** com validação
- 🗺️ **Roadmap** com evolução do projeto
- 📊 **Design responsivo** para todos os dispositivos

## 🚀 Tecnologia

- **React** - Biblioteca para construção de interfaces

- **Vite** - Build tool ultrarrápida
- **Tailwind CSS** - Framework de utilitários CSS

## 📦 Instalação

### Pré-requisitos
- Node.js 16+ 
- npm ou yarn

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/AlissonMarcoschalenge77/TDB-Responde.git

# Entre na pasta
cd TDB-Responde

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
